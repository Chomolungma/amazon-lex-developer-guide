# BuiltinIntentSlot<a name="API_BuiltinIntentSlot"></a>

Provides information about a slot used in a built\-in intent\.

## Contents<a name="API_BuiltinIntentSlot_Contents"></a>

 **name**   
A list of the slots defined for the intent\.  
Type: String  
Required: No
